# Workflows

n8n orchestration assets live here.

- `contracts/` for input and output contracts between workflow stages.
- `n8n/intraday/` for live alert workflows.
- `n8n/daily/` for end-of-day summary and publishing workflows.
- `n8n/shared/` for reusable workflow modules and shared logic references.

Keep workflows modular, idempotent, and easy to rerun safely.
