# App

Vue frontend application code lives here.

- `public/` holds static assets.
- `src/components/` holds reusable UI components.
- `src/layouts/` holds shared page layouts.
- `src/pages/` holds page-level route views.
- `src/router/` holds route definitions.
- `src/services/` holds frontend data access helpers.
- `src/styles/` holds global styling assets.
- `src/utils/` holds framework-agnostic frontend utilities.
