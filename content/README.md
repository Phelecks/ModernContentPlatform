# Content

Final editorial output stored in GitHub lives here.

- `topics/{topic_slug}/{date_key}/article.md` for final article content.
- `topics/{topic_slug}/{date_key}/summary.json` for structured daily summary output.
- `topics/{topic_slug}/{date_key}/metadata.json` for publish metadata.
- `topics/{topic_slug}/{date_key}/video.json` for YouTube embed and video metadata.
- `shared/` for reusable editorial fragments or shared assets if needed later.
